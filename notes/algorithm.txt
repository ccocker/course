Allocation Algorithm

1. Create a list of all preferences.
2. Filter all preferences to include only known tutors.
3. Sort all preferences by tutor (preference count) in ascending order
4. Sort all tutors preferences in ascending order
5. Create list of allocations.
   5a. REPEAT UNTIL all preferences have been added.
       i. FOR EACH tutor.
            Get next preference.
            Check if next preference is non consecutive.
            If non consecutive then add to allocation and remove preference.
            Else remove preference.
6. Process all allocations.
7. Allocate new staff.
